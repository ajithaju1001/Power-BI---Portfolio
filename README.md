# Productivity Log Dashboard.
👉 Click Here to View Live Dashboard 

🛠️ The Tech Stack
**Data Source**: Microsoft Lists (for real-time data entry)
**Automation**: Power Automate (Approval Workflow)
**Visualization**: Power BI


📝 Project Overview
In this project, Instead of raw data flowing directly into the report, I built an enterprise-grade pipeline that ensures every entry is verified by leadership.


⚙️ How it Works
Submission: Team members enter their daily output via a Microsoft List.
Validation: A Power Automate flow is triggered automatically once everyday with consolidated table of all new entries.
Approval: The supervisor receives a notification to Approve or Reject the entry.
Reporting: Only Approved entries are pulled into the Power BI dashboard, ensuring 100% data accuracy and management oversight.
